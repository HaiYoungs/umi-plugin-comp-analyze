# umi-plugin-comp-analyze

A umi plugin

## Install

```bash
pnpm i umi-plugin-comp-analyze
```

## Usage

Configure in `.umirc.ts`,

```js
export default {
  plugins: [
    ['umi-plugin-comp-analyze'],
  ],
}
```

Start Analyze Server

```js
umi analyzeComp
or
umi ac
```
